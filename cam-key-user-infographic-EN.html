ALLOTMENT TYPES & LIFECYCLE
📅 Schedule Allotment

Defined for a route/schedule period. Applies to multiple flights automatically.

Use case: Long-term customer agreements, seasonal capacity planning
✈️ Flight Allotment

Defined for a specific flight. One-time capacity reservation.

Use case: Spot bookings, special shipments, ad-hoc requests
🔒 Hard vs Soft Allotment

Hard: Guaranteed capacity, cannot be sold to others
Soft: Priority access, can be released if unused

Key: Hard allotment = contract commitment
