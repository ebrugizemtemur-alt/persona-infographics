EMBARGO vs RESTRICTION HIERARCHY
🚫 EMBARGO (First-Level Check)
External Authorities / Regulators
  • Countries, IATA, international bodies
  • Example: US embargo on Iran (only HUM allowed)
  • Associated with penalties/sanctions if violated
  • Highest priority in system hierarchy
  • Legal/compliance implications
⚡ System Behavior
If embargo exists → System stops immediately with popup (before restriction check)
⚠️ RESTRICTION (Second-Level Check)
Internal TK Operational Rules
  • Aircraft capacity limits, aircraft type restrictions
  • Operational constraints (e.g., summer routes)
  • Company-defined constraints
  • Station capabilities (no refrigeration, weekend closure)
  • DG acceptance limits per station
⚡ System Behavior
If no embargo → System proceeds to check restrictions (second popup)
