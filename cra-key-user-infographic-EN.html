WORKFLOW & DECISION POINTS
📥 OPR026 Execute → ⏱️ 5-min Job Import → 📋 CRA215 AWB List → 🔍 CRA212 Rate Audit → ✅ Rate Audited → 📊 CRA089 Bulk Finalize → 🏁 Finalized → 📤 CRA041 IATA Upload → 💼 SAP Transfer
🎯 Rate Audit Control
Primary check: chargeable weight consistency. Weight validation serves as proxy for total charge accuracy.
⚖️ Decision: Accept/Reject
If discrepancy found: Reject via CRA183 (Interline) or initiate CCA correction (CASS)
🔄 Reopen Handling
Post-operation changes (repack, services) require reopen. Rate decreases need pricing approval.
📅 Billing Period
IATA logic via execution date. Bi-monthly (15-day) cycle. Weekly possible for future flexibility.

8
PROCESS DOMAINS
🏠 Domestic CASS (TR Origin)
Cash, Credit Card, Credit Customer, Money Transfer. Station-specific invoicing. VAT complexity limits CCA usage.
🌍 International CASS
99%+ prepaid via CASS. No credit process. Weight validation is primary control. Country-specific rules apply.
✈️ Interline (Inward/Outward)
Partner airline settlements via ICH. SPA/SPOT/TAKT rates. Memo-based corrections (no CCA). Weekly IATA file exchange.
🔄 CCA Corrections
Post-invoice adjustments. Weight/rate updates, agent code changes. Limited domestic use due to VAT issues.
