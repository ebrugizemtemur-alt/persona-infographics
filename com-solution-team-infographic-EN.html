🔍 FRONT-CHECK API (TK Go)
How It Works:
  1. User clicks "Source Flight" in booking
  2. System calls Flight Information Service
  3. Service returns only eligible flights
  4. Filters out non-compliant flights
  5. User never sees error-level restrictions
Advantage: Seamless UX - users only see viable options
