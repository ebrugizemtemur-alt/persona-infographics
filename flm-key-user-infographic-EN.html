Ensure 100% data match between NetLine (schedule source) and COMMIS (booking system)

✓ Zero inconsistencies in daily diff report
✓ No duplicate records (same flight registered multiple times)
✓ No ghost flights (cancelled flights still appearing active)
✓ Timely handover to Operations at 72h before departure
