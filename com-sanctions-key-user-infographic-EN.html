JOBS TO BE DONE
🔍 Monitor Open Cases in Accuity
24/7 shift team reviews risky matches that appear as "open case". Analyze name/company similarity scores and determine true positives.
⚡ Real-time Booking Screening
Every booking flows to Accuity within 2-3 seconds. System automatically evaluates shipper, consignee, forwarder against sanction lists.
🚫 Block Sanctioned Shipments
When confirmed match found, block booking immediately. If cargo already in transit, coordinate return to origin point.
📋 Handle Receiver Changes
Critical pain point: receiver info can change even after cargo reaches destination. Re-screen and potentially stop shipment mid-journey.
📊 Risk Parameter Review
Evaluate similarity thresholds, false positive rates. Manual review required as no direct customer communication - system works automatically.
