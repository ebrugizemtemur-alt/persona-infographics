JOBS TO BE DONE
📋 Process Reservation Requests
Receive bookings via API, FFR, TK GO, and phone. Validate cargo details, routes, and special requirements.
🛡️ Embargo & Accuity Checks
Perform compliance checks for international shipments, embargo controls, and Accuity/sanction screening.
💰 ACRM Pricing Integration
Apply contract rates from ACRM (Agent Contract Rate Management) system for pricing calculations.
🔄 Routing & Allotment
Find optimal routing with available capacity. Manage allotments and blocking for key customers.
⏰ 6.5 Hour Constraint
Ensure all reservations are finalized minimum 6.5 hours before flight departure (Cut-off time).

PAIN POINTS MATRIX
Pain Point Description Business Impact Severity
Cut-off Time Pressure Must finalize all bookings 6.5 hours before departure Last-minute changes cause capacity loss CRITICAL
Accuity Manual Re-check System requires manual sanctions check for each shipper/consignee Delays booking confirmation, repetitive work HIGH
ACRM-COMIS Price Sync Contract rates sometimes don't pull correctly from ACRM Wrong pricing, revenue leakage, agent disputes HIGH
Multi-leg Routing Complexity Hub connections (IST-DOH-destination) need manual capacity check per leg Suboptimal routing, missed connections MEDIUM
API Error Handling Rejected API bookings require manual follow-up with agents Lost bookings, customer dissatisfaction MEDIUM
Allotment Visibility Difficult to see real-time allotment consumption per customer Overbooking or underutilization of contracts MEDIUM
